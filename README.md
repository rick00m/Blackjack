# Blackjack Web App

A browser based Blackjack game built with Python's Flask module. This project incorporates backend logic with simple frontend integration. 

## Features  
Blackjack gameplay:
    -Deck creation and shuffling
    -User can choose to hit or stand
    -Win/Loss/Push outcomes
    -User is able to stake and winnings/losses affect balance
-Web app interface built with HTML and CSS
