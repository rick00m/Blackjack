# Blackjack CLI

A terminal-based Blackjack game built with Python using a colorful, Vegas-style command-line interface. This project incorporates classic Blackjack gameplay logic with a rich, interactive terminal UI.

## Features

Blackjack gameplay:

- Deck creation and shuffling
- User can choose to hit or stand
- Win/Loss/Push outcomes
- User can stake and winnings/losses affect balance
- Vegas-style CLI with ASCII card art, chip stack display, and colored suits
- Animated card dealing and beginner-friendly interface

## Requirements

- Python 3.7+
- `rich` library
- `colorama` library

## Installation

```bash
pip install rich colorama
```

## Running the Game

```bash
python blackjack_vegas_cli.py
```
